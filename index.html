UI-Basics: Test One

Try to finish the exercises to the best of your abilities - read the tasks and instructions carefully!

You CAN use google, but you cannot ask teachers or other classmates for help - good luck!

Task Instructions

Starting up

  • Make and link a CSS stylesheet and work from there.

Content Styling

  • Align the text in the header to the center of the page.
  • Change the color of THIS list item to blue.
  • In the images section, make all of the images appear side-by-side.
  • Give three h3 elements in the html document the same class. Change color of the h3 elements to orange using rbg. The size of the text should be 2em and the font should be Arial. The text should appear as uppercase.
  • Select all descendant links in the footer. Change the links' color to green on visited and red on hover.

Box Model

  • In the "Box Model" section, add 5 pixels of padding to the section element that contains a paragraph and an image.
  • In the "Box Model" section, make sure the margins of the section element which contains a paragraph and an image is 10% of its parent's width.

Images & Links

  • In the "Link" section, insert a link that redirects the user to an image of your favourite book cover. Make sure that the image opens in a new tab.
  • In the "Images" section, add image tree.jpg from the image folder. Make the image appear rounded. Add this: https://caseytrees.org/tree-species/ link to this image.

Hierarchy

  • Move the footer in HTML to the bottom of the document.

Finishing Up

  • Once you have finished all the tasks, strike out the all the list items that are test instructions.
